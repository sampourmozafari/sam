No.1

Wordpress portfolio

Our first project of the semester was to create our first portfolio through the blogging system Wordpress. Besides a wordpress site we also had to make a HTML5 website which documented our first five days at Copenhagen Business Academy as newly enrolled multimedia design students.

No.2

International website

Our second project was a bit special. Besides being our first group project, we also had to group up with one from the MULB class and one from the international class. The objective was to make a travel website, which again had to be a Wordpress site, from where the international student was from, which in my case was Slovakia.

No.3

Political campaign

For our third project we had to make a campaign website for the 2016 United States presidential election. That ment we either had to choose Hillary Clinton or Donald Trump. My group chose Hillary. For the project we had to use HTML and CSS instead of the blogging system Wordpress. Besides making the site we also had to create a animated banner and document our choices along the way.

No.4

Intercultural website

This project was our first real world scenario assignment. Our objective was to create a website, again using HTML and CSS, for Pernille Sommer, an author who wanted a redesign of her current website. This was also our first project which had to be resposive.

No.5

New portfolio

The final project of the semester was a redesign of our original portfolio. This project is the result of all we have learned this first semester and what you now see is a result of that.
